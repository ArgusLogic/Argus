## Agent 自主降级能力 🏆

`scanme.nmap.org` 跑出 3 个亮点：

1. `whois_lookup` 两 provider 都失败 → **继续流程** 并在报告里注明原因
2. `dir_bruteforce` 超时 → **自动切 `http_request`** 挨个探 13 个常见路径（/admin/ /.git/ /backup/ /wp-admin/ 等），找出 `/images/` 目录遍历 + `/shared/` 403
3. `port_scan` 对 198.18.x.x 不可达返回空 → **自己推理**出"http 200 说明 80 实际开，只是扫描器网络受限"，在报告中明确说明

这种"工具失败不阻塞、自主换路子、结论可解释"的行为，是 Argus **自演化工具链设计**的核心价值证据。
